# odin4 Website

Professional and modern website for the odin4 tool - Samsung Flash Tool for Linux.

## Content

This website presents:

- **Overview:** Introduction to the tool and its benefits
- **Features:** Complete list of odin4 functionality
- **Safety:** Information about validation and device protection
- **Getting Started:** Step-by-step installation and usage instructions
- **Documentation:** Complete reference of command-line options and examples
- **Library:** C++ API documentation for integration
- **Components:** Explanation of supported firmware components
- **Contributing:** Links to GitHub repository and contribution guides

## Files

- `index.html` - Main website page
- `styles.css` - CSS styles with modern and responsive design
- `script.js` - Interactive JavaScript functionality
- `logo.png` - Official odin4 logo

## How to Use

1. Open the `index.html` file in a web browser
2. The website is fully responsive and works on desktop, tablet, and mobile
3. Use the navigation menu to access different sections
4. Click the "Copy" buttons on code blocks to copy commands

## Features

- Modern design with professional dark theme
- Fully responsive (mobile-first)
- Smooth and intuitive navigation
- Code blocks with functional copy button
- Smooth animations and transitions
- Optimized performance
- No placeholders or simulations - everything is real and functional
